# Request ID Tracing

Every request gets a unique ID that flows through all services.

## Generation

```typescript
import { randomUUID } from 'crypto';

function requestIdMiddleware(req: Request, res: Response, next: NextFunction) {
  const id = req.headers['x-request-id'] || `req_${randomUUID()}`;
  req.requestId = id;
  res.setHeader('X-Request-ID', id);
  next();
}
```

Accept the client's ID if provided (for end-to-end tracing), otherwise generate one.

## Propagation

When calling downstream services, pass the request ID:

```typescript
const response = await fetch('http://inventory-service/stock', {
  headers: { 'X-Request-ID': req.requestId },
});
```

## In Logs

Attach to every log line via pino child logger:

```typescript
const logger = pino();

app.use((req, res, next) => {
  req.log = logger.child({ requestId: req.requestId });
  next();
});
```

Now you can correlate all log lines for a single request across services by searching for the request ID.
