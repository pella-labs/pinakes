# Handling File Downloads via API

## Direct Download

```typescript
app.get('/files/:id/download', async (req, res) => {
  const file = await fileService.getById(req.params.id);
  if (!file) return res.status(404).json({ error: { code: 'NOT_FOUND' } });

  res.setHeader('Content-Type', file.mimeType);
  res.setHeader('Content-Disposition', `attachment; filename="${file.name}"`);
  res.setHeader('Content-Length', file.size);

  const stream = await fileService.getReadStream(file.storagePath);
  stream.pipe(res);
});
```

## Presigned URLs

For large files or when you want to offload to S3:

```typescript
app.get('/files/:id/download-url', async (req, res) => {
  const file = await fileService.getById(req.params.id);
  const url = await s3.getSignedUrl('getObject', {
    Bucket: 'my-bucket',
    Key: file.storagePath,
    Expires: 3600,
  });
  res.json({ download_url: url, expires_in: 3600 });
});
```

## Range Requests

Support `Range` header for resumable downloads:

```
Range: bytes=0-1023
→ 206 Partial Content
Content-Range: bytes 0-1023/5242880
```

Useful for large files over unreliable connections.
