# React Server Functions

## Beyond Server Actions

**Server Functions** are async functions that run on the server and can be called from both server and client components. Server Actions are a specific type of Server Function used in form actions and transitions.

## Defining Server Functions

```typescript
'use server';

export async function searchProducts(query: string) {
  const results = await db.product.findMany({
    where: { name: { contains: query } },
    take: 20,
  });
  return results;
}

export async function getRecommendations(productId: string) {
  return db.recommendation.findMany({
    where: { productId },
    include: { product: true },
  });
}
```

## Calling from Client Components

```tsx
'use client';
import { searchProducts } from './actions';

function SearchBar() {
  const [results, setResults] = useState([]);

  async function handleSearch(query: string) {
    const data = await searchProducts(query);
    setResults(data);
  }

  return (
    <div>
      <input onChange={(e) => handleSearch(e.target.value)} />
      <ResultsList results={results} />
    </div>
  );
}
```

## Security

Server functions are exposed as HTTP endpoints. Always validate inputs and check authorization. Never trust client-provided data.
