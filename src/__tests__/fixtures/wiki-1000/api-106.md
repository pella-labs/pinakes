# API Audit Logging

## What to Audit

Every **write operation** must be audit-logged:

- Who (user ID, API key, IP)
- What (resource type, resource ID, action)
- When (timestamp)
- How (method, path, request body summary)
- Result (success/failure, status code)

## Schema

```sql
CREATE TABLE audit_log (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  actor_id TEXT NOT NULL,
  actor_type TEXT NOT NULL, -- 'user', 'api_key', 'system'
  action TEXT NOT NULL,     -- 'create', 'update', 'delete'
  resource_type TEXT NOT NULL,
  resource_id TEXT NOT NULL,
  changes JSONB,            -- { field: { old: x, new: y } }
  ip_address INET,
  request_id TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_audit_actor ON audit_log(actor_id, created_at);
CREATE INDEX idx_audit_resource ON audit_log(resource_type, resource_id);
```

## Implementation

```typescript
async function auditLog(ctx: RequestContext, action: string, resourceType: string, resourceId: string, changes?: object) {
  await db.insert(auditLogTable).values({
    actorId: ctx.auth.userId,
    actorType: ctx.auth.type,
    action,
    resourceType,
    resourceId,
    changes: changes ? JSON.stringify(changes) : null,
    ipAddress: ctx.ip,
    requestId: ctx.requestId,
  });
}
```

## Retention

Keep audit logs for 2 years minimum. Archive to cold storage after 90 days. Never delete audit logs as part of a user deletion request — they're a legal requirement.
