# gRPC Health Checking Protocol

## Standard Service

gRPC has a built-in health checking protocol:

```protobuf
syntax = "proto3";

package grpc.health.v1;

service Health {
  rpc Check(HealthCheckRequest) returns (HealthCheckResponse);
  rpc Watch(HealthCheckRequest) returns (stream HealthCheckResponse);
}

message HealthCheckRequest {
  string service = 1;
}

message HealthCheckResponse {
  enum ServingStatus {
    UNKNOWN = 0;
    SERVING = 1;
    NOT_SERVING = 2;
  }
  ServingStatus status = 1;
}
```

## Implementation

```typescript
const healthImpl = {
  check: (call, callback) => {
    const service = call.request.service;
    
    if (service === '' || service === 'order-service') {
      const isHealthy = checkDependencies();
      callback(null, {
        status: isHealthy ? 'SERVING' : 'NOT_SERVING',
      });
    } else {
      callback({ code: grpc.status.NOT_FOUND });
    }
  },
};
```

## Kubernetes Integration

K8s natively supports gRPC health checks:

```yaml
livenessProbe:
  grpc:
    port: 50051
    service: ""
  periodSeconds: 10
```
