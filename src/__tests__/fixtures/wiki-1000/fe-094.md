# Clipboard API and Rich Text

## Modern Clipboard API

The **Clipboard API** provides async read/write access to the system clipboard:

```typescript
async function copyToClipboard(text: string) {
  try {
    await navigator.clipboard.writeText(text);
    showToast('Copied!');
  } catch {
    // Fallback for older browsers
    fallbackCopy(text);
  }
}

async function pasteFromClipboard() {
  const text = await navigator.clipboard.readText();
  return text;
}
```

## Rich Content

Copy HTML and images to the clipboard:

```typescript
async function copyRichContent(html: string, plainText: string) {
  const blob = new Blob([html], { type: 'text/html' });
  const textBlob = new Blob([plainText], { type: 'text/plain' });

  await navigator.clipboard.write([
    new ClipboardItem({
      'text/html': blob,
      'text/plain': textBlob,
    }),
  ]);
}
```

## Permission Model

The Clipboard API requires:
- HTTPS (or localhost)
- User gesture for read operations
- Focus on the page

## Use Cases

- Copy-to-clipboard buttons for code blocks
- Paste images from clipboard into an editor
- Cross-application data transfer
