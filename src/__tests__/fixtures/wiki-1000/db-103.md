# Elasticsearch Query DSL

## Query Context vs Filter Context

**Query context**: how well does this document match? Affects relevance score.
**Filter context**: does this document match yes/no? No scoring, faster, cacheable.

```json
{
  "query": {
    "bool": {
      "must": [
        { "match": { "title": "database performance" } }
      ],
      "filter": [
        { "term": { "status": "published" } },
        { "range": { "date": { "gte": "2024-01-01" } } }
      ]
    }
  }
}
```

## Important Query Types

- **match**: full-text search with analysis
- **term**: exact value match (keyword fields)
- **bool**: combine queries with must/should/must_not/filter
- **multi_match**: search across multiple fields
- **function_score**: custom scoring logic

## Aggregations

```json
{
  "size": 0,
  "aggs": {
    "by_category": {
      "terms": { "field": "category.keyword", "size": 10 },
      "aggs": {
        "avg_price": { "avg": { "field": "price" } }
      }
    }
  }
}
```
