# CSS Houdini and Custom Paint

## What Is Houdini

**CSS Houdini** is a set of low-level APIs that give developers access to the CSS engine. The most practical API today is the **Paint Worklet**, which lets you draw custom backgrounds with JavaScript.

## Custom Paint Example

```javascript
// paint-worklet.js
class CheckerPaint {
  static get inputProperties() {
    return ['--checker-size', '--checker-color'];
  }

  paint(ctx, size, props) {
    const cellSize = parseInt(props.get('--checker-size')) || 20;
    const color = props.get('--checker-color').toString() || '#000';

    for (let y = 0; y < size.height; y += cellSize) {
      for (let x = 0; x < size.width; x += cellSize) {
        if ((x / cellSize + y / cellSize) % 2 === 0) {
          ctx.fillStyle = color;
          ctx.fillRect(x, y, cellSize, cellSize);
        }
      }
    }
  }
}

registerPaint('checker', CheckerPaint);
```

```css
.element {
  --checker-size: 20;
  --checker-color: #e5e7eb;
  background-image: paint(checker);
}
```

## Browser Support

Paint Worklets are supported in Chromium browsers. Firefox and Safari support is limited. Use `@supports (background: paint(id))` for progressive enhancement.

## Practical Uses

Custom gradients, patterns, borders, and decorative effects that would otherwise require images or complex SVG. The paint is resolution-independent and GPU-accelerated.
