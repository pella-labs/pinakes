# React 19 Features Overview

## New Hooks

### useActionState

Replaces `useFormState` with a cleaner API for form actions:

```tsx
const [state, formAction, isPending] = useActionState(
  async (prevState, formData) => {
    const result = await submitForm(formData);
    return result;
  },
  initialState
);
```

### useOptimistic

Built-in optimistic update support:

```tsx
const [optimisticMessages, addOptimistic] = useOptimistic(
  messages,
  (state, newMessage) => [...state, { ...newMessage, sending: true }]
);
```

## ref as a Prop

No more `forwardRef` wrapper needed. `ref` is passed as a regular prop:

```tsx
function Input({ ref, ...props }) {
  return <input ref={ref} {...props} />;
}
```

## Document Metadata

Components can render `<title>`, `<meta>`, and `<link>` tags anywhere in the tree. React hoists them to `<head>` automatically.

## use() Hook

The `use` hook reads resources (promises, context) and can be called conditionally, unlike other hooks.

## Actions

Form submissions can call async functions directly. Combined with `useActionState` and `useFormStatus`, this provides a complete server interaction model without third-party libraries.

See [[fe-046]] for Server Actions and [[fe-043]] for ref-related patterns.
