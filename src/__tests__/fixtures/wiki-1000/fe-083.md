# Streaming and Progressive Loading

## Progressive Enhancement

A well-built web application works at every stage of loading. Content is readable as HTML arrives, styles apply progressively, and JavaScript enhances interactivity last.

## HTML Streaming

React 18's streaming SSR sends HTML in chunks:

1. Shell (layout, navigation) renders first
2. Content sections stream in as data resolves
3. Client-side JavaScript hydrates each section independently

Users see meaningful content within hundreds of milliseconds instead of waiting seconds for the full page.

## Progressive Image Loading

Show low-quality placeholders while full images load:

- **LQIP** (Low Quality Image Placeholder): blur-up from a tiny version
- **Dominant color**: show the image's average color
- **SQIP** (SVG-based): trace the image as a simplified SVG

## Streaming APIs

For large data sets, use streaming APIs to display results as they arrive:

```typescript
async function* streamResults(query: string) {
  const response = await fetch(`/api/search?q=${query}`);
  const reader = response.body!.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    yield JSON.parse(decoder.decode(value));
  }
}
```
