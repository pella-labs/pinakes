# OWASP Top 10 Mitigations

This page maps our defenses to the OWASP Top 10 (2021 edition). Each category links to the relevant internal documentation.

## A01: Broken Access Control

- RBAC enforced at gateway and service layer — see [[sec-003]]
- ABAC for fine-grained resource policies — see [[sec-004]]
- Automated authorization tests in CI that verify every endpoint requires auth
- IDOR prevention via ownership checks on all resource endpoints

## A02: Cryptographic Failures

- TLS 1.3 enforced on all external endpoints
- mTLS for internal service communication — see [[sec-005]]
- Encryption at rest for all databases — see sec-017
- Secrets never stored in code or config files — see sec-015

## A03: Injection

- Parameterized queries everywhere; ORMs configured to reject raw SQL
- Input validation at API boundaries using zod schemas
- CSP prevents script injection — see [[sec-007]]

## A04: Insecure Design

- Threat modeling required for new features (documented in [[arch-patterns]])
- Rate limiting on all authentication endpoints
- Account lockout after 5 failed attempts with exponential backoff

## A05: Security Misconfiguration

- Infrastructure as code with security baselines
- Automated scanning via [[deploy-pipeline]] pre-deploy checks
- Default-deny network policies
